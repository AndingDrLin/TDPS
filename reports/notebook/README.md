# Team 15-PTSD Lab Notebook Builder

This folder is dedicated to generating the line-following lab notebook as a DOCX file.

## Structure

- scripts/notebook_content.py: English notebook content source (4 entries).
- scripts/generate_notebook_figures.py: Creates PNG charts from notebook data.
- scripts/build_notebook_docx.py: Builds the final DOCX and embeds generated figures.
- scripts/export_notebook_json.py: Exports notebook content as structured JSON cells.
- scripts/build_all.py: One-command build pipeline.
- assets/images/: Generated chart images.
- outputs/: Final DOCX artifact.

## Build

From repository root:

```bash
python reports/notebook/scripts/build_all.py
```

Expected output:

- `reports/notebook/outputs/team15_ptsd_lab_notebook.docx`
- `reports/notebook/outputs/team15_ptsd_lab_notebook.json`

## Notes

- Notebook entries are intentionally realistic: each page includes failure-analysis-improvement loops.
- All technical symbols are mapped to real code in `TDPS/` and `TDPS/simulator/`.
- Figures are generated by code first, then embedded into DOCX.
