# TDPS Project

> 基于 STM32 的课程项目：巡线行驶 + LoRa 通信 + 雷达避障

## 项目目标

- **任务 1：巡线行驶**：沿黑线稳定通过弯道与路口，到达终点。
- **任务 2：无线通信**：在指定拱门区域通过 LoRa 发送队伍与计时信息。
- **任务 3：避障通行**：使用 24 GHz 及以上雷达判断障碍左右并选择空侧通过。
